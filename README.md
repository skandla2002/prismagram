# prismagram

Instargram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret(Login)
- [ ] Log in
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] edit the photo (Delete)
- [ ] See the feed
